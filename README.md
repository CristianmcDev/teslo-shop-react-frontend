# Teslo Shop

Proyecto fullstack realizado como práctica siguiendo un curso de React de Fernando Herrera en Udemy, enfocado en simular una tienda online moderna y aprender el flujo completo de desarrollo web.

## 🛠️ Tecnologías utilizadas

* React 19
* TypeScript
* Vite
* React Router
* Tailwind CSS
* Zustand (estado global)
* TanStack Query (gestión de servidor y cache)
* React Hook Form (formularios)
* Axios (peticiones HTTP)
* Axios (peticiones HTTP)
  

**UI / Estilos:**

* Radix UI
* Lucide React (iconos)
* Sonner (notificaciones)
* Tailwind utilities (clsx, tailwind-merge, class-variance-authority)

**Herramientas de desarrollo:**

* ESLint
* TypeScript ESLint
* Vite plugin React (SWC)



## 🧠 Conceptos y funcionalidades trabajadas

* Navegación y sistema de rutas con React Router
* Manejo de Query Parameters y filtros dinámicos
* Persistencia de sesión y autenticación con JWT
* Protección y autorización de rutas
* Gestión de estado global con Zustand
* Manejo de datos asíncronos con TanStack Query
* Cache, invalidación y mutations
* Paginación y filtrado de productos
* Formularios y validaciones avanzadas
* Integración frontend/backend mediante APIs REST
* Transformación e integración de código generado con IA
* Despliegue completo de frontend, backend y base de datos en la nube

## 🛠️ Funcionalidades implementadas

* Autenticación de usuarios
* Login persistente
* CRUD de productos
* Gestión de pedidos
* Panel de productos con filtros y paginación
* Formularios validados
* Subida y gestión de imágenes
* Despliegue fullstack en producción


## ⚠️ Notas

Este proyecto tiene fines educativos y de práctica. Algunas funcionalidades pueden estar simplificadas y no representar un entorno de producción real.
